# 小说网系统安装指南

## 系统要求

### 服务器环境
- PHP 7.4 或更高版本
- MySQL 5.7 或更高版本
- Apache 或 Nginx Web服务器
- 支持 URL 重写（mod_rewrite）

### PHP扩展要求
- PDO MySQL扩展
- JSON扩展
- CURL扩展
- Fileinfo扩展（可选）

## 快速安装

### 第一步：上传文件
1. 将项目所有文件上传到您的网站根目录
2. 确保以下目录有写入权限：
   - `/backend/config/`
   - `/public/uploads/`（如果存在）

### 第二步：访问安装页面
1. 在浏览器中访问您的网站域名
2. 系统会自动跳转到安装向导页面

### 第三步：完成安装步骤

#### 步骤1：环境检查
- 系统会自动检查PHP版本和所需扩展
- 如果检查通过，点击"下一步"

#### 步骤2：数据库配置
- 数据库主机：通常是 `localhost`
- 数据库端口：通常是 `3306`
- 数据库名称：创建新的数据库名
- 数据库用户名：MySQL用户名
- 数据库密码：MySQL密码

#### 步骤3：管理员设置
- 网站名称：您的网站显示名称
- 管理员姓名：管理员真实姓名
- 管理员密码：设置管理员登录密码

#### 步骤4：安装完成
- 系统会显示安装成功信息
- 保存管理员账号信息

## 详细配置

### 数据库配置
安装完成后，数据库配置文件位于：`/backend/config/database.php`

```php
<?php
return [
    'host' => 'localhost',
    'port' => '3306',
    'database' => 'xiaoshuowang',
    'username' => 'root',
    'password' => 'your_password',
    'charset' => 'utf8mb4'
];
```

### 网站配置
系统配置存储在数据库的 `sys_config` 表中，可以通过后台管理界面修改。

## 后台管理

### 登录后台
- 后台地址：`您的域名/admin/`
- 默认账号：`admin`
- 密码：安装时设置的密码

### 后台功能
- 小说管理：添加、编辑、删除小说
- 分类管理：管理小说分类
- 用户管理：管理注册用户
- 系统设置：配置网站基本信息
- AI设置：配置AI接口参数

## 故障排除

### 常见问题

**1. 安装页面无法访问**
- 检查文件是否完整上传
- 检查Web服务器配置
- 确认PHP版本符合要求

**2. 数据库连接失败**
- 检查数据库配置信息
- 确认数据库服务正常运行
- 检查数据库用户权限

**3. 页面显示404错误**
- 启用Apache的mod_rewrite模块
- 检查Nginx的rewrite配置
- 确认.htaccess文件存在

**4. 文件上传失败**
- 检查上传目录权限
- 检查PHP上传限制配置
- 确认磁盘空间充足

### 日志查看
系统操作日志存储在数据库的 `sys_log` 表中，可以通过SQL查询查看详细错误信息。

## 安全建议

1. **定期备份**：定期备份数据库和重要文件
2. **更新系统**：及时更新PHP和MySQL版本
3. **权限控制**：合理设置文件和目录权限
4. **密码安全**：使用强密码并定期更换
5. **HTTPS加密**：启用SSL证书保护数据传输
