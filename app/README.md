# 小说网系统

## 项目简介
一个完整的小说阅读和创作平台，支持AI小说生成和微信公众号对接。

## 功能特性
- 小说阅读与分类管理
- 用户管理和会员系统
- AI小说智能生成
- 微信公众号对接
- 快速安装部署
- 多平台兼容（H5、小程序、App）

## 技术栈
- 后端：PHP 7.4+
- 前端：uni-app + Vue.js
- 数据库：MySQL 5.7+
- 部署：Apache/Nginx

## 快速开始
1. 上传源码到网站目录
2. 访问网站自动进入安装页面
3. 配置数据库信息和管理员账号
4. 完成安装开始使用

## 目录结构
```
├── backend/          # PHP后端API
├── frontend/         # uni-app前端
├── install/          # 安装系统
├── database/         # 数据库文件
├── config/           # 配置文件
└── public/           # 公共资源
```
