========================================
    小说网系统 - 文件清单
========================================

【部署方式】
由于自动打包失败，请手动复制以下文件到服务器：

【必须上传的目录和文件】

1. app/ 目录下的所有文件：
   - index.php
   - .htaccess
   - README.md
   - FILES.md
   - GENERATED.md
   - 部署说明.txt

2. app/backend/ 目录
   - 包含所有PHP框架文件
   - config/database.php (需要修改配置)

3. app/admin/ 目录
   - index.php
   - login.html
   - login.php
   - dashboard.html
   - content.php
   - category.php
   - users.php
   - settings.php
   - ai_settings.php
   - backup.php
   - init_tables.php

4. app/install/ 目录
   - index.php

5. app/public/ 目录
   - index.html
   - uploads/ 目录

6. app/database/ 目录
   - structure.sql

7. app/runtime/ 目录
   - cache/ 目录
   - logs/ 目录
   - backups/ 目录

【部署步骤】

1. 在服务器上创建网站根目录
2. 上传上述所有文件和目录
3. 修改 backend/config/database.php 中的数据库配置
4. 设置 runtime/ 和 public/uploads/ 目录权限为755
5. 访问 http://您的域名/admin/init_tables.php 初始化数据库表
6. 访问 http://您的域名/admin/ 登录后台

【后台功能地址】
- 系统设置：/admin/settings.php
- AI设置：/admin/ai_settings.php
- 数据库备份：/admin/backup.php
- 用户管理：/admin/users.php
- 内容管理：/admin/content.php
- 分类管理：/admin/category.php

【注意事项】
1. 必须先修改数据库配置
2. 必须运行 init_tables.php 初始化表
3. 确保PHP版本 >= 7.4
4. 确保MySQL版本 >= 5.7
