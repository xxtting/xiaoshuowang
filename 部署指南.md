## 🚀 快速部署步骤

### 第一步：上传文件

1. 将 `xiaoshuowang_app_final.zip` 上传到服务器网站根目录
2. 解压文件到当前目录

```bash
# Linux/Mac
unzip xiaoshuowang_app_final.zip

# Windows
# 使用解压工具或右键解压
```

### 第二步：配置数据库

编辑文件：`backend/config/database.php`

```php
<?php
// 修改以下配置为您的数据库信息
define('DB_HOST', 'localhost');        // 数据库主机
define('DB_PORT', '3306');             // 数据库端口
define('DB_NAME', 'xiaoshuowang');     // 数据库名
define('DB_USER', 'root');             // 数据库用户名
define('DB_PASS', 'your_password');    // 数据库密码
define('DB_CHARSET', 'utf8mb4');
define('DB_PREFIX', '');
```

### 第三步：设置目录权限

```bash
# Linux/Mac
chmod -R 755 runtime/
chmod -R 755 public/uploads/
chmod -R 755 runtime/backups/

# Windows
# 右键目录 -> 属性 -> 安全 -> 设置权限
```

### 第四步：初始化数据库表

访问网址：`http://您的域名/admin/init_tables.php`

点击初始化按钮，自动创建所需的数据库表。

### 第五步：访问网站

- **前台首页**: `http://您的域名/`
- **后台管理**: `http://您的域名/admin/`
- **安装向导**: `http://您的域名/install/`（首次安装）

## 🔧 常见问题

### 1. 数据库连接失败

**问题**: 提示数据库连接错误

**解决**:
1. 检查 `backend/config/database.php` 配置
2. 确认数据库服务器可访问
3. 确认数据库用户有权限

### 2. 页面显示404

**问题**: 访问后台页面显示404

**解决**:
1. 检查文件是否完整上传
2. 检查 Apache/Nginx 重写规则
3. 确认 `.htaccess` 文件存在

### 3. 无法上传图片

**问题**: 图片上传失败

**解决**:
1. 检查 `public/uploads/` 目录权限
2. 检查 PHP 上传大小限制
3. 检查磁盘空间

### 4. 功能页面空白

**问题**: 设置页面显示空白

**解决**:
1. 访问 `/admin/init_tables.php` 初始化数据库表
2. 检查 PHP 错误日志
3. 确认数据库配置正确
